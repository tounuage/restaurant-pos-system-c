# Restaurant POS System (C)

## 📌 Overview
This project is a console-based Point of Sale (POS) system developed in C. It simulates order processing and financial tracking for a restaurant environment.

## ⚙️ Features
- Menu-based navigation system
- Handles multiple customer orders
- Allows continuous item entry with quantity input
- Calculates subtotal and final total with discount/tax
- Generates end-of-day summary report (total revenue, average order value)

## 🛠️ Technologies Used
- C programming language
- Standard input/output
- Control structures (loops, if-else)
- User-defined functions

## 🧠 What I Learned
- Modular programming using functions
- Managing multiple orders using loops
- Implementing real-world logic (POS system)
- Financial calculations and reporting

## ▶️ How to Run
1. Compile:
   gcc main.c -o pos
2. Run:
   ./pos
